# What is the use of Css ?

# How many types of Css ? 
- Three types of Css :- 
  - Inline  Css <p style="font-size:20px"></p>
  - Internal Css <style>
      p{
        background-color:"red"
      }
  </style>
  - External Css // Writing css seperately
